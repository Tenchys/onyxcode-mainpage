# Landing page OnyxCode.cl

## Descripción
OnyxCode.cl es una startup de desarrollo de software web personalizado. La landing page transmite nuestra visión y motiva a potenciales clientes a contactarnos.

## Directrices de diseño
- Diseño responsivo, accesible (para lectores de pantalla) y fácil de navegar.
- Paleta de colores inspirada en el onice con toque moderno.
- Secciones: Header, Hero, Servicios, Contacto (formulario sin conectar), Footer.

## Desarrollo
- React con Tailwind CSS, optimizado para carga rápida.
- Imágenes en formato WebP.
